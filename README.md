# 🐾 Southampton Veterinary Clinic

A fully functional veterinary clinic website built as a 
front-end web development project.

🔗 **Live site:** https://rockayam.github.io/Southampton-Veterinary-Clinic-Project/index.html

## Features
- 10-page multi-section website
- Appointment booking form with date validation and 
  localStorage persistence
- Pet records system with add, search, and delete functionality
- Contact form with client-side validation
- Vaccination schedule toggle (Dogs / Cats)
- Pharmacy category filter (All / Preventative / Care)
- Emergency numbers show/hide toggle
- Scroll-reveal animations using IntersectionObserver
- Smooth page transition fade effects

## Built With
- HTML5
- CSS3 (Flexbox, Grid, custom styling)
- JavaScript (DOM manipulation, localStorage, 
  IntersectionObserver API)

## Pages
| Page | Description |
|------|-------------|
| Home | Hero section with featured services |
| About | Clinic background and values |
| Services | Full list of veterinary services |
| Team | Staff profiles |
| Booking | Appointment booking with form validation |
| Records | Add, search and delete pet medical records |
| Vaccinations | Vaccine schedules for dogs and cats |
| Pharmacy | Filterable medication catalogue |
| Emergency | Emergency contacts with toggle display |
| Contact | Validated contact form |

## Project Context
Built as part of the Web Development module at the 
University of Southampton (2025–2026).
